# Configuration files for ESPHome devices

## esphome-web-1d95ee.yaml

Controller for a custom-built Wind & Weather indicator built around a vintage Danforth/White Wind Velocity indicator and a pair of LEDs mimicking Boston's [Hancock tower's weather beacon](https://secretboston.co/old-hancock-towers-weather-guide/). Relies on a connection to a Home Assistant instance to get weather data and set parameters.

CAD files for the stand model here: https://www.printables.com/model/1614635-stand-for-danforthwhite-wind-indicator
